# Contributing to ai-skills-zh

感谢你考虑为本仓库贡献技能！🙌

## 📋 贡献前准备

### 技能规范
- 文件夹名称使用 kebab-case（如 `my-skill`）
- 必需文件：`SKILL.md`（包含 YAML 前置信息）
- 可选目录：`scripts/`、`references/`、`assets/`

### 测试技能
```bash
# 1. 创建符号链接
ln -s /path/to/your/skill ~/.claude/skills/your-skill

# 2. 重启 AI Agent

# 3. 测试技能功能
```

## 📝 提交 PR 流程

1. Fork 仓库
2. 创建分支：`git checkout -b feature/your-skill-name`
3. 添加你的技能
4. 更新文档（README.md、skills-list.md）
5. 推送并创建 PR
